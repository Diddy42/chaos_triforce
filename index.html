What's going on?

The base game (with n=3):

1. Take three starting points in a plane
2. Randomly select any point and consider that your current position
3. Randomly select any one of the three starting points
4. Move half the distance from your current position to the selected point
5. Plot the current position
6. Repeat from step 3
7. After a lot of points a pattern starts to emerge, the Sierpiński triangle (with the 3 starting points as vertexes)
